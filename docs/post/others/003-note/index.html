# linux  date和时间戳转换
1：查看当前时间
date +"%Y-%m-%d %H:%M:%S"

输出："2021-03-25 18:12:38"

2：查看指定时间，转为时间戳
date +%s
date -d "2021-03-25 18:12:38" +"%Y-%m-%d %H:%M:%S"
date -d "2021-03-25 18:12:38" +%s

3：时间戳转换为date
date -d @1416387827
date -d @1416387827 +"%Y-%m-%d %H:%M:%S"
